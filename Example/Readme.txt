This is the code that accompanies the task - each of the steps that is explained in the task was used to create this porject.

To see how this code works:
1. Copy mySite.rar to your local machine and extract the files to a folder called mysite
2. From the command line, navigate to the folder you extracted these files too
3. In the command line type "python manage.py runserver"
4. Navigate to "http://127.0.0.1:8000/webapp/" in your browser
